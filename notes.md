# Python

## 1. Basic data types in python

1. Numeric types:
   - represents numbers
   - integers
        1
        123456
        2
        3
        -10
        -198
   - floats/decimals
        0.33
        67.899
        -56.988

2. String types:
   - alpha-numeric characters
   - Examples:
        'Alex Malla'
        "Alex Malla"
        "12345"
        "Alex95@(*&(*^*^))"


3. Boolean type:
    - True OR False


4. None type: it represents nothing
